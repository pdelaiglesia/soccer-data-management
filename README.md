# soccer-data-management
Proyecto en C++ que carga, gestiona y consulta datos de fútbol a partir de archivos CSV. Permite buscar clubes, mostrar información por ligas, ordenar equipos, consultar jugadores, visualizar alineaciones y navegar por un menú interactivo. Incluye procesamiento de millones de registros y modelado mediante clases como Club, Player y gameLineup.
